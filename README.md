# MapArt Commissions

A webpage documenting how I structure and sell **Minecraft map art commissions** for players on the [**MineSuperior**](https://minesuperior.com/) Skyblock server.

🌐 **View the page:**  
https://niladoze.github.io/mapart/

---

## About

This page explains the system I used to sell and organize **map art commissions** for players on the MineSuperior Skyblock server.

In Minecraft, map art is created by carefully placing blocks so that an image appears when viewed through an in-game map. These artworks are often used as decorative pieces on Skyblock islands.

The site serves as a **reference page for my commission process**, showing:

- how commissions were structured  
- examples of map art work  
- the ordering format used for players  
- pricing and request details  

---

## Purpose

The page was created to give players a **clear overview of how to request custom map art**, while also showing previous work and explaining the process behind each commission.

Instead of handling requests entirely through chat, this page allowed players to quickly understand:

- what map art commissions were available  
- how to order one  
- what information they needed to provide
